# RestauranteApp

Sistema de gestión para restaurante desarrollado en Java y MySQL.

## Tecnologías utilizadas
- Java
- MySQL
- Apache NetBeans

## Funcionalidades
- CRUD de productos
- Gestión de inventario
- Administración de empleados
- Gestión de reservas
- Gestión de pedidos

## Objetivo
Proyecto académico enfocado en Programación Orientada a Objetos y bases de datos.
